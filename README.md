# Nagri Handwritten Character Recognition

A deep learning project for recognizing handwritten **Devanagari (Nagri)** characters using a **Convolutional Neural Network (CNN)** trained on the publicly available Devanagari dataset.

---

## Project Overview
This project aims to classify handwritten Devanagari characters into their respective categories using image-based deep learning techniques.

It was implemented and trained on **Google Colab** using **TensorFlow / Keras**.

---

## Key Features
- Dataset: Devanagari Character Dataset (from Kaggle)  
- Model: Convolutional Neural Network (CNN)  
- Accuracy: ~99% on training and validation  
- Tools: Google Colab, TensorFlow, NumPy, Matplotlib  
- Input: 32x32 grayscale character images  
- Output: Predicted Devanagari character label
